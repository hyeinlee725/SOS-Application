### 요약
- 사용자는 응급 상황시 연락을 취하고자 하는 사람의 전화번호를 등록한다.
- 사용자는 응급 상황시 보내고자 하는 메시지를 사전에 등록한다.
- SOS버튼을 2번 누를 시 보내고자 하는 메시지와 사용자의 현재 위치 정보가 담긴 지도가 전송된다.
